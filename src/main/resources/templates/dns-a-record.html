DNS A Records

DNS A records ('address' records) are the most basic type of record. They identify an (IPv4) IP address with a particular hostname.

E.g.
Here we can see dnsmonks.com resolves to 18.188.64.96.

% dig dnsmonks.com A +short
18.188.64.96
%
Typically a hostname will have a single IP address but it is also possible to associate several addresses with a single hostname to provide redundancy or load balancing.
