DNS AAAA Records

DNS AAAA (typically pronounced 'Quad A') records are similar to A records but provide IPv6 addresses for a hostname instead of IPv4.

E.g.
Here we can see the AAAA record for icann.org:

% dig icann.org aaaa +short
2001:500:88:200::7
%
Like A records, a domain can have multiple AAAA records for a single hostname.
